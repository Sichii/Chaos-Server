Class FixedSet<T>

Namespace
Chaos.Collections.Specialized
Assembly
Chaos.Collections.dll

Represents a thread-safe fixed-size set of items. If the set is at capacity and a new item is added, the oldest item is removed.

public sealed class FixedSet<T> : ICollection<T> where T : notnull

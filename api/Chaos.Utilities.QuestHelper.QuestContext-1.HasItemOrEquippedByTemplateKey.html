HasItemOrEquippedByTemplateKey(string, int)

True if Source's Inventory has at least count of templateKey, or Source's Equipment contains at least one item with that template key. Use for deliverables that can be turned in either still in the bag or already worn (e.g. a quest bouquet that the player equipped before reaching the turn-in NPC).

public bool HasItemOrEquippedByTemplateKey(string templateKey, int count)

Parameters

templateKey string
count int

Returns

bool

Remarks

Equipment slots are non-stacking, so the equipment side only contributes 1 toward the requirement. For count > 1, the inventory side must satisfy the full count on its own.

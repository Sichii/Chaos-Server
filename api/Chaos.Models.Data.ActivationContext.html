TargetDirection

The direction of the target. If the target is a creature, this will be the direction the creature is currently facing. If the target is a point, this will be the relative direction from the source to the target.

TargetMap

The target map of the activation. If the target is a creature, this will be the map the creature is currently on. Otherwise, it will be the map the target was on at time of activation.

TargetPoint

The target point of the activation. If the target is a creature, this will be the point the creature is currently at.

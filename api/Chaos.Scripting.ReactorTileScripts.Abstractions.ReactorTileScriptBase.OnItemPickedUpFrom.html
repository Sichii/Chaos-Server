OnItemPickedUpFrom(Aisling, GroundItem, int)

Triggers when an item is picked up from the reactor tile by an Aisling

public virtual void OnItemPickedUpFrom(Aisling source, GroundItem groundItem, int originalCount)

Parameters

source Aisling

The creature that picked up the item

groundItem GroundItem

The ground item that got picked up

originalCount int

The count of the item before it may have been merged into a stack

Remarks

Comes after OnPickup(Aisling, Item, int)
When an item is picked up, it may be absorbed into an existing stack in the inventory. When this happens, this method is called with the original ground item and it's count as reference. The stack that got merged into is not available.

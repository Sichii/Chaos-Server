Class TailSamplingProcessor

Namespace
Chaos.Definitions
Assembly
Chaos.dll

A tail-sampling processor that buffers spans per trace and makes trace-level export decisions. When a root span ends, if any span in the trace was ratio-sampled or exceeded its duration threshold, the entire trace (all buffered spans) is exported. Otherwise, the trace is discarded silently. This processor should be added before the exporter in the pipeline.

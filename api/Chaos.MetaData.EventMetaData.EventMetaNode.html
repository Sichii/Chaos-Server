Remarks

events follow a structure like this

"01_start"
"01_title"

  • {QuestName}
    "01_id"
  • {??} -- icon? or maybe legend mark id?
    "01_qual"
  • {circle}
  • {classflag}
    "01_sum"
  • {Quest Summary}
    "01_result"
  • {??} -- icon after completing?
    "01_sub"
  • {maybe something to do with required pre-req quest}
    "01_reward"
  • {QuestRewards}
    "01_end"

Properties

Id

The id of the event. To have the event marked as completed, you will need a legend mark with a key equal to this id.

Name

The name of the event

Page

The page that this event is on

PrerequisiteEventId

The id of the event that must be completed before this event can be started

QualifyingCircles

The circles (level ranges) that this event is available to

Result

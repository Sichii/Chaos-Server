Properties

AccountBound

Whether or not the item is account bound. If the item is account bound, it cannot be traded or dropped

ArmorUsesOvercoatSprites

When this item is equipped, it will be considered an overcoat for display purposes. This allows items in the normal armor slot to use overcoat sprites and be dyeable. If an overcoat is equipped, that item will show instead.

BuyCost

The amount of gold it costs to buy this item from a merchant

Category

The category of the item, used for bank or shop sorting

Color

Defaults to None(lavender)
The color of the item

DisplaySprite

Default null
If specified, this is the sprite value used to display the item on the character when equipped

EquipmentType

Default null
If specified, this is the type of equipment this item is

Gender

Default null
If specified, this is the gender required to use this item

IsDyeable

Whether or not the item can be dyed. This determines whether a color change applied to an item persists through serialization and deserialization. Town dyes on pre-99 armor is not dye, theyre completely different sprites.

IsModifiable

Whether or not an instance of the item can be modified. This controls if an item can be affected by meta node mutators.

MaxDurability

Defaults to null
If specified, the max durability of the item

MaxStacks

The maximum number of this item that can be in a stack. If set to 1, the item will not be stackable

Modifiers

Defaults to null
If specified, these are the stats this item grants when equipped

NoTrade

Whether or not the item can be traded. If the item is not tradeable, it cannot be traded to other players via exchange or dropping. These items are still droppable, but are locked to the player that dropped it.

OvercoatUsesArmorSprites

When this item is equipped, it will be considered an armor for display purposes. This allows items in the overcoat slot to use normal armor sprites. If both an armor and an overcoat are equipped, the overcoat will display.

OverridesBootsSprite

When this item is equipped, the boots sprite will be set to 0

OverridesHeadSprite

When this item is equipped, the head sprite will be set to 0

PantsColor

Default null
If specified, this armor will have pants, and they will be this color

PreventBanking

Whether or not the item can be banked

SellValue

The amount of gold given for selling this item to a merchant

Weight

The weight of the item in the inventory, or equipped

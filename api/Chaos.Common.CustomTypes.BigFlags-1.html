Class BigFlags<TMarker>

Namespace
Chaos.Common.CustomTypes
Assembly
Chaos.Common.dll

Abstract base class for defining named flag constants with automatic bit index assignment. Derive from this class and define static fields of type BigFlagsValue<TMarker>. Fields will be automatically assigned sequential bit indices starting from 0. Use [BitIndex(n)] attribute to explicitly specify a bit index.

public abstract class BigFlags<TMarker> where TMarker : class

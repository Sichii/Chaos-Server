CurrentStage
NeverStarted

True if Source's Trackers.Enums has no value stored for TStage. Distinct from CurrentStage == default, which is also true when default(TStage) is the explicitly stored value.

Award amount experience to Source via the active experience distribution script (resolved through FunctionalScriptRegistry).

GiveGamePoints(int)

Add amount game points to Source. There is no cap or distribution script — game points are a settable integer counter on Aisling.

GiveGold(int)

Add amount gold to Source via TryGiveGold. The chain continues even if it fails (e.g. would exceed MaxGoldHeld); use a guard if you need to halt on failure.

GiveItem(string, int)

Create a fresh item of templateKey via IItemFactory, set its Count if greater than 1, and hand it to Source via GiveItemOrSendToBank.

GiveLegendMark(string, string, MarkIcon, MarkColor)

Add a new LegendMark with Now to Source's Legend, or accumulate the count if a mark with the same key already exists.

GiveSkill(string)

Resolve ISkillFactory from Services, create a skill of templateKey, and learn it via ComplexActionHelper.LearnSkill (which checks SkillBook capacity).

GiveSpell(string, byte?)

Resolve ISpellFactory from Services, create a spell of templateKey, and learn it via ComplexActionHelper.LearnSpell (which checks SpellBook capacity). If page is provided and the spell was learned, swap it into that slot. If the spell wasn't learned (e.g., capacity full), the page placement is silently skipped — quest rewards never gate on placement success.

GroupTeleport(string, IPoint)

Teleport every member of Source's group to point on the resolved map. If Source has no group, only Source is teleported.

HasActiveCooldown(string, out TimeSpan)

True if Source's Trackers.TimedEvents has an active (non-completed) event keyed by key. When true, remaining is set to the event's remaining time; otherwise it is Zero.

HasAllFlags<TMarker>(BigFlagsValue<TMarker>)

True if every bit in combined is set on Source's Trackers.BigFlags.

HasAllFlags<TFlag>(TFlag)

Check whether Source's Trackers.Flags contains the given enum flag.

HasGold(int)

True if Source's Gold is at least amount. Pure read — does not emit any client message.

HasItem(string, int)

True if Source's Inventory contains at least count of the item identified by templateKey.

HasItemOrEquipped(string, int)

True if Source's Inventory has at least count of templateKey, or Source's Equipment contains at least one item with that template key. Use for deliverables that can be turned in either still in the bag or already worn (e.g. a quest bouquet that the player equipped before reaching the turn-in NPC).

IncrementCounter(string, int)

Increment Source's Trackers.Counters for key by by.

RemoveSkill(string)

Remove the skill matching templateKey from Source's SkillBook.

RemoveSpell(string)

Remove the spell matching templateKey from Source's SpellBook.

SetFlag<TMarker>(BigFlagsValue<TMarker>)

Set the given big flag on Source's Trackers.BigFlags.

SetFlag<TFlag>(TFlag)

Set the given enum flag on Source's Trackers.Flags.

StartCooldown(string, TimeSpan)

Start an auto-consuming timed event keyed by key on Source's Trackers.TimedEvents. Re-addable: completed AutoConsume events with the same key are pre-cleaned by TimedEventCollection.AddEvent.

Teleport(string, IPoint)

Resolve the destination map via ISimpleCache<TResult> and traverse Source to point on it.

TryConsumeGold(int)

Attempt to remove amount gold from Source via TryTakeGold. Returns false if Source does not have enough; otherwise removes it and returns true. On the failure path, TryTakeGold sends a system "not enough gold" orange-bar message; gate with HasGold(int) first if a quest-specific reply is preferred.

TryConsumeItem(string, int)

Attempt to remove count of the item identified by templateKey from Source's Inventory. Returns false if the source does not have enough; otherwise removes them and returns true.

TryConsumeItemOrEquipped(string, int)

Consume that prefers the inventory and falls back to Equipment when count is 1. Returns false (consuming nothing) if neither side can satisfy the requirement.

WhenAt(TStage)

True if Source's Trackers.Enums currently holds the given stage value.

OnDisplaying(Aisling)

Called when the dialog is about to be displayed, but has not yet been displayed. You can use this event to modify the dialog before it is displayed, such as injecting text parameters into the dialog's text

OnNext(Aisling, byte?)

Called when the player responds to the dialog, whether by clicking Next, selecting an option, or making any kind of selection
If an option is clicked, the optionIndex will be the index of the option that was clicked
If a shop selection was clicked, the text of that option will be in the MenuArgs property of the dialog
If a selection was made from the player's inventory, spellbook, or skillbook, the Slot that was selected will be in the MenuArgs property of the dialog

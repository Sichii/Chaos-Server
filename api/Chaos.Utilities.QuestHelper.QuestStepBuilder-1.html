Fluent builder for an operation chain. Recorded operations are functions over QuestContext returning bool — true = continue the chain, false = halt (guard failed).

Remarks

Most guard methods (When, WhenNeverStarted, RequireFlag, RequireItem, etc.) take an optional failureReply parameter. When provided, a failed guard sends a Dialog Reply with the message and halts; when omitted, the guard halts silently.

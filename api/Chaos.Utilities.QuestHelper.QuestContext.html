Properties

OptionIndex

1-based index of the dialog option the player selected when advancing the dialog (e.g. menu choice or button option). Set by the dispatcher only when the lifecycle phase is Next; null in all other phases. Always null when the player advances a non-menu dialog (Normal, TextEntry, etc.) — Dialog.Next normalizes those to null before invoking the script.

Services

Service provider used to lazy-resolve framework dependencies (IItemFactory, etc.). The dispatcher (QuestDialogScript) is responsible for passing this through. Tests can use MockServiceProvider from Chaos.Testing.Infrastructure.

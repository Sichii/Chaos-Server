Properties

Contextual

Whether or not this dialog requires context passed to it from the previous dialog

IllustrationIndex

Index into the NPC's list of illustration filename variants. The client maintains this list by merging npci.tbl (inside npcbase.dat , client-side only — the server has no control over it) with the NPCIllust metafile the server pushes at login. npci.tbl entries occupy the low indices, metafile entries are appended after them.

For virtually every NPC exactly one filename is defined, so 0 is the correct value here almost always. Only set this to a non-zero value if the NPC actually has multiple illustration variants defined in one of those two sources, which is rare.

NextDialogKey

Defaults to null
If specified and the Type is Normal, the dialog will have a next button that will take them to the dialog with this template key

Options

A collection of options that can be selected from this dialog that lead to other dialogs.

PrevDialogKey

Defaults to null
If specified and the Type is not Normal, the dialog will have a previous button that will take them to the dialog with this template key

ScriptKeys

A collection of names of dialog scripts to attach to this dialog by default. The name of a script is just the name of the class without "Script" at the end.

ScriptVars

A collection of key-value pairs of key-value pairs
Each script that has variables needs a scriptName-Value pair, and the value of that entry is a dictionary of propertyName-Value pairs

TemplateKey

A unique id specific to this template. This must match the file name

Text

The text displayed on the dialog. This can be a string format with injectable parameters, but those parameters must be injected by a script

TextBoxLength

Defaults to null
If specified and the Type is DialogTextEntry, this will limit the length of the input text box

TextBoxPrompt

Defaults to null
If specified and the Type is DialogTextEntry, this will be the message displayed above the text box

Type

The type of dialog this is. This controls how the dialog is displayed.
